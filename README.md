## 시작 가이드
---
### Requirements
For building and running the application you need:
- Java 17 이상
- Spring Boot 3.x
- MySQL 8.0 이상
- Gradle 7.x 이상

### Installation
```bash
$ git clone https://github.com/Line4Thon-Nugu/Nugu-Backend.git
```
### Service Address
```
https://nugu-line4.vercel.app/
```
